CivicSuite / CivicPlan

Cited planning policy support that keeps humans responsible.

CivicPlan is the comprehensive-plan policy lookup and cited planning analysis support module for CivicSuite.

v0.1.0 planning policy foundation

Current State

The package shell, API foundation, deterministic plan-policy lookup, consistency support, staff-analysis outline, records-ready export checklist, and public sample UI exist. Official planning determinations, legal advice, live GIS, live LLM calls, plan document ingestion, permitting-system integrations, and production staff-review queues are not shipped yet.

Product Promise

  • Find adopted plan-policy context with citations.
  • Surface sample consistency factors for staff review.
  • Draft cited staff-analysis outlines without making official determinations.
  • Preserve source/proposal/reviewer provenance for municipal records.
